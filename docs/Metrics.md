# Metrics

## Unified CX Intelligence System — Metrics Framework

This file defines the KPI and metric model for evaluating system performance, CX intelligence quality, operational responsiveness, and governance adherence.

---

# 1. System Health Metrics
- Workflow success rate
- Workflow failure rate
- Integration uptime (Slack, Asana, social platforms)
- Credential health (expired/invalid tokens)
- HealthCheck results

---

# 2. Signal Intelligence Metrics
- Total inbound events
- Events per platform
- Events per severity tier
- Sentiment distribution
- Topic/category frequency

---

# 3. Risk & Incident Metrics
- Average incident score
- High/critical events per week
- Recurrence trends
- False-positive alerts
- Human override count

---

# 4. EWS Performance Metrics
- Time to acknowledge alerts
- Time to human decision
- Alerts created (per platform)
- Approved vs. rejected alerts

---

# 5. Content Workflow Metrics
- Time from approval → Asana task creation
- Time from task creation → draft completion
- Time from draft → approval
- Tasks requiring revisions
- Reviewer workload

---

# 6. Governance & Safety Metrics
- Zero auto-publishing violations
- Logging completeness
- Least-privilege credential check
- Human-approval compliance

---

# 7. Business Impact Indicators (Derived)
- Sentiment trends over time
- Early detection indicators
- Thematic clustering
- Operational bottlenecks

---
